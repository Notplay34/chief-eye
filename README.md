# РегДок

Внутренняя система для двух павильонов автоуслуг возле МРЭО: оформление документов, касса, изготовление номеров, склад и управленческая аналитика.

Стек проекта: `FastAPI`, `SQLAlchemy`, `PostgreSQL`, статический `HTML/CSS/JS` frontend, `nginx` и `systemd` на сервере.

## Документация

В проекте поддерживаются два основных документа:

- [Проектная документация](docs/PROJECT.md) — назначение системы, роли, бизнес-процессы, кассы и пользовательские сценарии.
- [Техническая документация](docs/TECHNICAL.md) — структура репозитория, локальный запуск, миграции, деплой, бэкапы, проверки и безопасность.

Если документация и код расходятся, исправлять нужно документацию или код в том же изменении. Не заводите отдельные черновики, runbook'и и заметки как новые источники правды.

## Быстрый старт

```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

```bash
cd frontend
python3 -m http.server 8080
```

Открыть: `http://127.0.0.1:8080/login.html`.

## Проверка перед релизом
